# Substance Use & Treatment Facility Analysis

## Overview
This project analyzes substance use trends and treatment facility access across the United States using publicly available datasets. The goal is to identify gaps between substance use rates and available treatment resources at a regional level.

## Business Problem
Access to treatment facilities does not always align with areas experiencing high substance use rates. This project explores whether geographic disparities exist between substance use prevalence and treatment availability.

## Data Sources
- SAMHSA (Substance Abuse and Mental Health Services Administration)
- CDC (Centers for Disease Control and Prevention)
- NCHS (National Center for Health Statistics)
- Kaggle public datasets

## Tools & Technologies
- Python (Pandas, NumPy)
- Data Cleaning & Transformation
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

## Approach
- Collected and merged datasets from multiple public health sources  
- Cleaned and standardized data for consistency across sources  
- Aggregated data at state and county levels  
- Analyzed trends in substance use and treatment facility distribution  
- Created visualizations to compare regional differences  

## Key Findings
- Identified regions with high substance use rates but limited treatment facility access  
- Observed inconsistencies in facility distribution relative to population needs  
- Highlighted areas where resource allocation may be insufficient
